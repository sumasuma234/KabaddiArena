Kabaddi Arena 🏆

Kabaddi Arena is an Android application developed using Kotlin and XML in Android Studio. The app is designed to manage Kabaddi match scores and display match results in a simple and user-friendly interface.

📱 Features

- Welcome Splash Screen
- Player Name Entry
- Raid Success Counter
- Tackle Success Counter
- Total Score Calculation
- Result Display Screen
- Reset and Undo Functions
- Attractive UI Design

🛠 Technologies Used

- Kotlin
- XML
- Android Studio
- Firebase (Optional Backend)

📂 Project Structure

- "MainActivity.kt" → Main scoring logic
- "ResultActivity.kt" → Result display screen
- "activity_main.xml" → Main UI design
- "activity_result.xml" → Result page layout
- "SplashActivity.kt" → Welcome screen

🚀 Future Enhancements

- Firebase Database Integration
- Match History Storage
- Player Leaderboard
- User Login System
- Live Match Updates

🎯 Objective

The objective of this project is to simplify Kabaddi score management digitally instead of maintaining manual records.
